# Kazuma Hashiguchi

> 業務フローにAIを組み込み、実務で使えるアプリとして設計・実装しています。

生成AIを活用した業務アプリ開発に取り組んでいます。  
特に、**AIを業務フローへどう組み込むか** を意識し、UIだけでなく、保存・履歴管理・比較・運用まで含めたアプリ設計を行っています。

## 🚀 Portfolio

### 1. AI問い合わせ対応支援アプリ（実務特化）
業務フローにAIを組み込み、人間確認・承認・監査まで一貫して扱える問い合わせ対応アプリです。

- AIによるカテゴリ分類、優先度判定、要約、回答案生成
- 人間確認フロー、承認、差し戻し
- 担当者アサイン、SLA管理、通知、監査ログ
- 技術: Next.js / TypeScript / Prisma / SQLite / OpenAI API

Repository: https://github.com/qiaokouhezhen11-star/ai-support-workflow

---

### 2. AI業務計画アシスタント（構造化AI）
業務依頼文をもとに、AIがタスク分解・優先度・リスク・確認事項を構造化して生成するアプリです。

- 構造化出力による再利用しやすい計画生成
- フォローアップ質問による精度向上
- ローカルナレッジ参照による文脈強化
- 保存、再編集、比較、ダッシュボード表示
- 技術: Next.js / TypeScript / Prisma / SQLite / OpenAI API

Repository: https://github.com/qiaokouhezhen11-star/ai-task-planner

---

### 3. AIチャットアプリ（業務利用を意識したチャット）
ストリーミング応答とファイル参照に対応した、業務利用を意識したAIチャットアプリです。

- ChatGPTライクなUI
- `.txt` / `.md` ファイル参照
- ストリーミング応答、停止、再送、続きを生成
- 会話履歴の保存、検索、切り替え
- 技術: React / React Router / JavaScript / OpenAI API

Repository: https://github.com/qiaokouhezhen11-star/ai-chat-app

---

## 💡 Strengths

- UI / API / DB をまたいで、**フルスタックでアプリを形にできる**
