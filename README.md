# はじめまして

生成AIエンジニアへのキャリアチェンジを目指して学習している橋口和真です。  
React / Next.js / TypeScript / OpenAI API を中心に、生成AIを実務に組み込む Web アプリを制作しています。

## 取り組んでいること

- 生成AIアプリ開発の学習
- フルスタック構成でのポートフォリオ制作
- 業務で使える AI 体験の設計
- 実装だけでなく、保存・運用・比較まで含めたアプリ設計

## 主なポートフォリオ

### AI業務計画アシスタント

業務依頼文を入力すると、生成AIが以下を構造化して整理する実務向けアプリです。

- 依頼内容の要約
- タスク分解
- 優先順位
- 実行ステップ
- 想定リスク
- 確認事項

さらに、RAG によるナレッジ参照、フォローアップ質問、履歴保存、比較表示、ステータス管理、ダッシュボードまで実装しています。

- Repository: https://github.com/qiaokouhezhen11-star/ai-task-planner

### AIチャットアプリ

ChatGPTライクなUIで、ストリーミング応答とファイル参照に対応した生成AIチャットアプリです。  
`.txt` / `.md` ファイルをアップロードし、その内容をもとにAIへ質問できる機能を実装しています。

- Repository: https://github.com/qiaokouhezhen11-star/ai-chat-app
